**********   steps for putting a sublime snippet for typing console.log   **********

1. add 

{ "keys": ["ctrl+super+c"], "command": "insert_snippet", "args": { "name": "Packages/Colorsublime-Themes/console-log-all-the-things.sublime-snippet" } } 

to Sublime > Preferences > Key Bindings - User


2.
<snippet>
    <content><![CDATA[console.log('${1:Placeholder}', ${1:Placeholder});]]></content>
    <!-- Optional: Scope the tab trigger will be active in -->
    <scope>source.javascript</scope>
    <!-- Optional: Description to show in the menu -->
    <description>console.log('allTheThings', allTheThings);</description>
</snippet> 

in a file called "console-log-all-the-things.sublime-snippet"


3. put "console-log-all-the-things.sublime-snippet" in Sublime > Preferences > Browse Packages… > Colorsublime-Themes
